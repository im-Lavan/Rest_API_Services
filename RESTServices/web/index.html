CycleNest REST API

RESTful Web Service for Sustainable Item Rental Platform

Welcome to CycleNest API Documentation

CycleNest is a RESTful web service developed for the COMP30231 Service-Centric and Cloud Computing coursework at Nottingham Trent University. It enables sustainable item rental by connecting people who have unused items with those who need them temporarily. This API powers a sharing economy platform where users can browse available items, calculate distances, apply filters, and create rental requests.

Azure Cosmos DB OSRM API Azure VM JSON Java Tomcat Git Hub NetBeans Jmeter

Local Development

localhost:8080 - For local testing and development

Distance Calculation

GET Get distance between two coordinates
http://localhost:8080/RESTServices/webresources/RESTAPI/direct?startLon=-0.1276&startLat=51.5074&endLon=-2.2426&endLat=53.4808
GET Get distance for specific item
http://localhost:8080/RESTServices/webresources/RESTAPI/items/i002/distance?userLat=51.5074&userLon=-0.1276

Item Retrieval

GET Get all items (raw, no distance)
http://localhost:8080/RESTServices/webresources/RESTAPI/items
GET Get all items with distance calculation
http://localhost:8080/RESTServices/webresources/RESTAPI/items?userLat=51.5074&userLon=-0.1276

Pagination

GET Page 1 with distance calculation
http://localhost:8080/RESTServices/webresources/RESTAPI/items?page=1&userLat=51.5074&userLon=-0.1276
GET Page 2 without distance calculation
http://localhost:8080/RESTServices/webresources/RESTAPI/items?page=2

Filtering

GET Filter by category
http://localhost:8080/RESTServices/webresources/RESTAPI/items?category=Tools
GET Filter by city
http://localhost:8080/RESTServices/webresources/RESTAPI/items?city=Nottingham
GET Filter by condition
http://localhost:8080/RESTServices/webresources/RESTAPI/items?condition=Excellent
GET Multiple filters (category + condition)
http://localhost:8080/RESTServices/webresources/RESTAPI/items?category=Tools&condition=Good
GET Filter + pagination
http://localhost:8080/RESTServices/webresources/RESTAPI/items?city=London&page=1
GET Filter with distance calculation
http://localhost:8080/RESTServices/webresources/RESTAPI/items?condition=Excellent&userLat=51.5074&userLon=-0.1276
GET All filters + pagination + distance
http://localhost:8080/RESTServices/webresources/RESTAPI/items?category=music&city=oxford&condition=excellent&page=1&userLat=53.3811&userLon=-1.4701

Rental Requests

POST Request item with user ID
curl -X POST "http://localhost:8080/RESTServices/webresources/RESTAPI/items/i001/request?user_id=Alice"
POST Request item (default user)
curl -X POST "http://localhost:8080/RESTServices/webresources/RESTAPI/items/i001/request"
PUT Cancel request by ID
curl -X PUT "http://localhost:8080/RESTServices/webresources/RESTAPI/requests/REQ-1768234228723-4362/cancel"
